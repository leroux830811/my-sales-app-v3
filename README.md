# Deli Sales Pro

This is a CRM application for sales representatives in the food and deli industry, built with Next.js and Firebase.

## Core Features

- **Sales Dashboard**: A comprehensive overview of sales performance.
- **Interaction Tracking**: Log calls, meetings, and competitor activity.
- **Route Planning**: Plan daily customer visits.
- **Order Management**: Place new orders and process stock returns.
- **Product Catalog**: View and manage the product list.
- **Reminders**: Set follow-up reminders.
- **Photo Gallery**: Take and store photos for storefronts or interactions.
- **Reporting**: Generate and export reports in various formats.

## Getting Started

The main entry point for the application is `src/app/page.tsx`. The app is structured using modern React patterns, including hooks and contexts for state management.
