# **App Name**: Deli Sales Pro

## Core Features:

- Sales Dashboard: Dashboard view of sales performance, customer interactions, and upcoming tasks.
- Interaction Tracking: Ability to log and track interactions with customers (calls, emails, meetings).
- Reminder System: Functionality to set reminders for follow-ups with clients.
- Product Catalog: Store product information like pricing, descriptions, and available stock.
- Email Template Generation: AI-powered tool to generate personalized email templates for customer outreach, based on sales and past orders.
- Data Export: CSV Export for Reporting

## Style Guidelines:

- Primary color: Muted mustard yellow (#D4A373) to convey trustworthiness and expertise in the food industry.
- Background color: Off-white (#F5F5DC), providing a clean and neutral backdrop that focuses attention on the content and is suited for a light scheme.
- Accent color: Terracotta (#E07A5F) for a touch of warmth in key interactions.
- Body and headline font: 'PT Sans', a modern and slightly warm sans-serif.
- Use clean, minimalist icons to represent different actions and data points.
- Prioritize clear information hierarchy and easy navigation for a user-friendly experience.
- Subtle animations for feedback on user interactions (e.g., form submissions, task completions).
